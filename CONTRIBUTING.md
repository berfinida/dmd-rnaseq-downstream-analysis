# Contributing

Thanks for your interest in contributing.

## Setup
```bash
pip install -r requirements.txt
```

## Run Scripts
```bash
python downstream_analysis/plot_volcano_like.py
python downstream_analysis/plot_heatmap.py
python downstream_analysis/plot_pca.py
python downstream_analysis/plot_summary_panel.py
streamlit run app.py
```

## Pull Requests
1. Create a feature branch.
2. Keep changes focused and documented.
3. Run quality checks before opening PR.
4. Open a PR with a clear summary of updates.

## Data Policy
Please do not add raw FASTQ files or other large raw sequencing files to this repository.
