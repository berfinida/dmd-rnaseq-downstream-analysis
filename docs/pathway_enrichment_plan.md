# Pathway Enrichment Plan (Not Yet Implemented)

Pathway enrichment is not currently implemented in this repository.

## What Would Be Required
- A reliable gene-level identifier mapping (for example Ensembl transcript -> gene symbol -> Entrez/Ensembl gene IDs).
- A clearly defined ranked/selected gene list from a statistically sound workflow.
- Real pathway databases and APIs.

## Why Annotation Is Required
Most enrichment tools require gene-level identifiers (gene symbols, Entrez IDs, or Ensembl gene IDs), not transcript IDs alone.

## Possible Future Tools
- gseapy
- Enrichr
- GO term enrichment
- KEGG pathway enrichment
- Reactome pathway enrichment

## Scientific Scope Note
Until real annotation and enrichment workflows are implemented, no pathway-level claims should be made.
